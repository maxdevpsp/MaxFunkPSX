P1 Controls:
Square   (Left Arrow)           - D
Cross    (Down Arrow, Confirm)  - F
Triangle (Up Arrow)             - J
Circle   (Right Arrow, Back)    - K

DPad (All Arrows) - Arrow keys

Return - Start
Right Shift - Select
